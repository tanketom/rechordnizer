Rechordnizer

Real-time piano chord recognition from microphone input. Play a chord, see it identified instantly.

▶ Open Rechordnizer

Runs in your browser. No install needed.

Features

Real-time Detection

Chord analysis at ~10 Hz using FFT-based chroma extraction with cosine similarity matching.

144 Chord Templates

12 root notes across 12 chord types: major, minor, 7th, maj7, m7, dim, aug, sus2, sus4, dim7, m(maj7), aug7.

Piano Visualization

Active notes highlighted on a keyboard display, with a chroma bar showing pitch class energy distribution.

Zero Install

Runs entirely in the browser using the Web Audio API. No downloads, no plugins, no dependencies.

How It Works

1

Microphone audio is captured via the Web Audio API

2

An AnalyserNode computes the FFT magnitude spectrum

3

FFT bins are mapped to 12 pitch classes (C through B) using the MIDI frequency formula

4

The chroma vector is matched against 144 pre-computed chord templates using cosine similarity

5

A majority-vote smoother stabilizes predictions over a sliding window
